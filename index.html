This will be the home page of the Loyc Core Libraries, a set of useful libraries for many kinds of .NET developers, with a special focus on collection classes, interfaces, adapters, and extension methods.

  • Loyc.Essentials fills in gaps in the .NET Base Class Library
  • Loyc.Collections adds sophisticated data structures-
  • Loyc.Syntax contains the LES parser and other parsing-related services including base classes for LLLPG.
